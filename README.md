<h1 align="center">Крестики - нолики 3x3</h1>
ASP.NET (C#) REST API для игры "Крестики - нолики 3x3" для двух игроков. Все игровые данные сохраняются в файлике. Игра написана на C# и использует веб-API ASP.NET
<h2 align="center">Описание API</h2>
Игроки по очереди расставляют свои метки (X или O) на поле. Первый игрок, который поставит три метки подряд (по горизонтали, вертикали или диагонали), побеждает в игре.
